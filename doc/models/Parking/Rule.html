Overview

Model to handle parking rules, this rules are associated to each parking lot, which means that it will be a copy (for each value on the enum :name) for every parking lot. This rules can be turn off/on on the dashboard project, by going to https://dashboard.telesoftmobile.com/dashboard/parking_lots/:id/rules (Change the :id value)

Table's Columns

  • name => [integer] Rule name (determined by the enum :name)
  • description => [text] Rule description (determined by the enum :name before saving)
  • status => [boolean] If the rule is on or off
  • agency_id => [bigint] Reference ID to which agency will be notified when this rule happens
  • lot_id => [bigint] Reference ID to which parking lot admin will be notified
  • created_at => [datetime]
  • updated_at => [datetime]
